About the Equality Matters webpage

What is the website about?

Equality Matters is place to access a range of measures and analyse them in relation to equality groupings.
There are over 200 measure pages available to explore

You will be able to get to a measure page through various pathways:

  • Search using the bar in the header
  • By Section 75 Category
  • By Government Department

Features:

  • Measure page covering current year data and time trends where available
  • Download image files of the current year data
  • Access automatically generated data summaries of each measures trends and values
  • Access and copy tabular data of each chart
  • Links to Data Portal Sources and the original publication where this measure was reported

Sources of data

All of the data presented is sourced from the NISRA Data Portal.

The NISRA Data Portal

The NISRA Data Portal is a repository for data on our population, society and economy.

The Data Portal is continually refreshed with new and updated datasets from across the NICS and Public Sector. It gives users the opportunity to view and filter data sets, plot interactive charts and visualise data on interactive maps, save queries, create favourite datasets and widgets, access data in multiple formats - xlsx, csv, JSON-stat, px and share results and automate processes using API queries.

Training Materials (NISRA Data Portal user guide, instructional video and FAQ section) have been published to help you navigate and use the new system.

You can sign-up to attend online training sessions by emailing info@nisra.gov.uk to register your interest. Further help, if needed, will also be available on request.

Rounding

For the purposes of this application, all values have been round to 0 decimal places.
